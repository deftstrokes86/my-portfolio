# **App Name**: RadPortfolio

## Core Features:

- PWA Support: PWA (Progressive Web App) support for offline access and installability.
- Accessibility Controls: Accessibility controls in the header menu: theme toggle, font size adjustment, high contrast toggle, reduce motion toggle, dyslexic-friendly font option, pause/stop animations button.
- Sticky Header: Header with sticky behavior, blur backdrop, logo/monogram, navigation links (Work, Case Studies, About, Writing, Contact), and a CTA button (Hire Me).
- Interactive Cards: Card component with image, metadata, tags, and call to action elements, with a subtle hover lift animation.
- SEO Optimization: Optimized SEO with semantic HTML, sitemap.xml, robots.txt, Open Graph, Twitter cards, and JSON-LD for Person and CreativeWork.
- Testimonials Section: Display user-provided testimonials in an elegant and engaging manner.
- Semantic Search: Smart search component to help the user find what they want in the blog and documentation, which may utilize a tool for advanced semantic understanding.

## Style Guidelines:

- Primary color: AccentA (#8A5CF6), a vibrant purple, suggesting sophistication.
- Background color: Light theme (#F7F5F2), a very light desaturated yellow. Dark theme: (#0B0D0E), a dark desaturated blue.
- Accent color: AccentB (#7AE7FF), a light, high-contrast blue for interactive elements.
- Display font: 'Fraunces Variable' (serif) for headings.
- Body/UI font: 'Inter Variable' (sans-serif) for body text and interface elements.
- Use a spacing grid of 4px and a maximum container width of 1200px with fluid gutters.
- Incorporate microinteractions such as a subtle hover lift on cards (translateY 4px) and link underlines that grow from the center.
